# mypro

## Build Setup

``` bash
# install dependencies
yarn install

# 编译部署 将会编译部署到 ccf.portal\modules\mypro 下
npm run deploy

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
